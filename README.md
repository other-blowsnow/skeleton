# skeleton
一个快速的骨架生成，基于页面原结构生成
动画效果

## 快速使用
```
// 使用前请为 body 设置属性 visibility: hidden   否则会闪屏
// 需要渲染的元素列表
var skeleton = new Skeleton(document.getElementsByClassName('skeleton-item'));

// 调用渲染
skeleton.render();

// 调用销毁
skeleton.destroy();
```
